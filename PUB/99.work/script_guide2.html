롯데손해보험 SCRIPT TAB/TOGGLE GUIDE

토글/탭 작업 가이드

기본정보

  • UI 디스플레이 관련 플러그인
  • 사용 : 탭, 토글, 아코디언 등 UI 디스플레이 open/close 컨트롤 사용

옵션정보

옵션정보
옵션 내용 기타
toggle_type 'toggle' / 'tab' / 'tooltip' default : 'toggle' 토글타입
selector 토글 컨텐츠 영역 셀렉터
selector_group true / false default : true 토글그룹속성 : 오픈시 동일형제요소 토글속성 초기화 여부
selector_btn '[data-toggle="btn"]' 토글이벤트 발생객체 셀렉터
selector_con '[data-toggle="content"]' 컨텐츠객체 셀렉터
selector_close '[data-toggle="btn-close"]' 토글 닫기기능 셀렉터 : selector_btn 옵션이외에 닫기 기능담당
event_flag true / false default : false 클릭이벤트실행시 e.preventDefault(); 적용여부
doc_event true / false default : false 외부영역 클릭시 닫기 설정
class_active 'active' 오픈시 추가 클래스
auto_scroll true / false default : false 오픈시 해당영역 자동스크롤 이동기능
callback_before null function() 콜백함수 : 팝업 오픈 전 실행
callback_load null function() 콜백함수 : 팝업 오픈 후 실행
callback_after null function() 콜백함수 : 팝업 닫기 시 실행

<script>
$plugin.togglecon('.tab-sub1', {
  toggle_type : 'tab',
  selector : '.tab-sub1 > ul > li',
  selector_btn : '> a',
  selector_con : '#href',
  auto_scrolltab : true //가로스크롤이 필요할 경우
});
</script>

탭 UI WAI-ARIA 역할 및 속성

탭 UI WAI-ARIA 역할 및 속성
구분 속성 설명
역할 [role="tablist"] 탭 메뉴를 그룹핑 하는 요소 / 탭 집합
[role="tab"] 각각의 탭메뉴 요소
[role="tabpanel"] 각 탭이 컨트롤 하는 탭 섹션
속성 [aria-selected="true/false"] [role="tab"] 의 선택상태
[aria-expanded="true/false"] [role="tabpanel"] 의 펼침상태

스크립트호출

<script>
//기본(그룹)
$plugin.togglecon('.area-toggle2 > li');

//개별
$plugin.togglecon('.area-toggle1', {
  selector_group : false
});

//내부닫기버튼
$plugin.togglecon('.area-toggle1', {
  selector_close : '[data-toggle="close"]',
});

//외부호출 : 열기
$plugin.togglecon('.area-toggle1').outputOpen();

//외부호출 : 닫기
$plugin.togglecon('.area-toggle1').outputClose();
</script>
