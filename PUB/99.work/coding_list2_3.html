퍼블리싱 작업공유

  • 로컬에서 /html/99.work/coding_list1.html 을 통해서도 열람가능.
  • 작업목록의 완료일이 표기된 페이지의 경우 개발진행가능.
  • 완료일 이후 수정발생시 수정일에 날짜가 표기되며, 수정일을 클릭하면 상세수정내역 확인이 가능(소스상에는 해당수정일로 주석검색)

스크립트1 : 레이어팝업

  • 레이어는 별도 페이지 퍼블리싱 작업하며, 호출샘플도 함께 제공
  • 참고 : /html/99.work/script_guide1.html

스크립트2 : 탭, 아코디언

  • 탭, 토글등 공통 UI 스크립트는 [data-autoset="toggle"] [data-autoset="tab"] 등 [data-autoset] 속성을 통해 퍼블리싱에서 공통선언
  • 별도 개발에서 제어할 경우 [data-autoset] 을 제거한 후 별도 선언하여 사용.
  • 참고 : /html/99.work/script_guide2.html

기타 : 디스플레이

  • 개발 조건에 따라 일부 컨텐츠를 숨김처리해야할 경우 인라인스타일로 display:none 처리하는것 보다는 none 클래스 추가/삭제를 통하여 디스플레이 조정
  • display none 의 경우는 다시 보여줘야 할 경우, display 속성은 inline, block, inline-block, table, flex 등 해당객체 UI에 따라 다양하게 변경되므로, 가급적 none 클래스로 제어한다.
  • 클래스로 제어할 경우 간격조정등 변동되는 ui컨트롤가능
