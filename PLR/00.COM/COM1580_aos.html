원가드 앱 설치

STEP 3

영업지원 앱 설치 후, 보안 모듈(원가드)을 설치하면 바로 사용하실 수 있어요.

  1. 1. 원가드 설치 바로가기 (설정 전) 영업지원 앱 로그인 완료 후 등장하는 원가드 설치를 진행해주세요.

  2. 2. 설정선택 설정에서 출처허용을 진행해주세요.

  3. 3. 원가드 설치 바로가기 (설정 후) 무시하고 설치하기를 선택해 설치를진행해주세요.

    중요!!) 확인버튼을 누르시면 안돼요! 설치가 안됩니다.

    세부정보 더보기를 꼭 탭 하신 후 무시하고 설치 확인 버튼을 꼭 눌러야 설치가 완료돼요.

  4. 4. 약관동의 및 설정하기 동의하고 시작하기 및 모든 설정 한번에 하기를 선택해주세요.

  5. 5. 배터리 사용량 최적화 중지 백그라운드 사용을 위해 허용을 선택해주세요.

  6. 6. KNOX 약관동의 동의체크 후, 동의를 선택해주세요.

  7. 7. 로그인 원가드 설치 완료 확인 후 로그인 해주세요.

    원가드 설치 완료 후 정책이 활성화되면 홈화면에서 영업지원 앱을 실행해주세요.
